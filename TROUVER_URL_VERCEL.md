# Comment trouver votre URL Vercel

Votre projet est connecté au dépôt GitHub : `nadaabbassi-a11y/myrent`

## Méthode 1 : Via le Dashboard Vercel

1. Allez sur https://vercel.com
2. Connectez-vous avec votre compte
3. Cliquez sur votre projet "myrent"
4. L'URL de production sera affichée en haut de la page
   - Format typique : `https://myrent.vercel.app`
   - Ou : `https://myrent-[hash].vercel.app`

## Méthode 2 : Via la ligne de commande

```bash
# Si vous avez Vercel CLI installé
vercel ls

# Ou pour voir les détails du projet
vercel inspect
```

## Méthode 3 : Vérifier les déploiements GitHub

1. Allez sur https://github.com/nadaabbassi-a11y/myrent
2. Cliquez sur l'onglet "Actions" (si activé)
3. Les déploiements Vercel apparaîtront avec l'URL

## URL probable

Basé sur votre nom de projet, votre URL est probablement :
- **https://myrent.vercel.app**

## Pour installer l'app iPhone

1. Ouvrez Safari sur iPhone
2. Allez sur votre URL Vercel
3. Appuyez sur le bouton de partage
4. Sélectionnez "Sur l'écran d'accueil"
5. L'app sera installée !

## Note importante

Assurez-vous que votre site est bien déployé et accessible en HTTPS (Vercel le fournit automatiquement). Les PWA nécessitent HTTPS pour fonctionner.
